# Creative Brainstormer Agent

Generates ideas, explores angles, and helps break through creative blocks.

## Usage

Describe the challenge or area where you need ideas. Optionally set `quantity` and `style` to shape the brainstorm.

## Examples

- "Name ideas for a productivity app aimed at freelancers" + quantity: 20
- "Campaign concepts for a summer product launch" + style: bold
- "Ways to improve onboarding for new employees" + style: practical
