# Strategic Advisor Agent

Helps think through decisions, weigh trade-offs, and build structured arguments.

## Usage

Describe the decision or situation you are facing. Optionally set `framework` and `constraints` to guide the analysis.

## Examples

- "Should we build this feature in-house or buy a vendor solution?" + framework: pros-cons
- "Evaluate three pricing models for our SaaS product" + constraints: must keep free tier
- "Help me prioritise this backlog of 10 initiatives for next quarter"
