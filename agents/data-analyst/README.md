# Data Analyst Agent

Interprets data, spots patterns, and delivers clear insights with actionable takeaways.

## Usage

Provide raw data, a dataset description, or numbers you want analysed. Optionally set `goal` and `output` to focus the analysis.

## Examples

- Paste a CSV of monthly sales figures to get trend analysis and anomalies
- "Compare Q3 vs Q4 conversion rates" + goal: identify drop-off causes
- "Summarise this survey data into key themes" + output: executive summary
