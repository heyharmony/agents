# Data Analyst

## Description

Analyses data sets, metrics, or figures to surface patterns, trends, and actionable insights presented in a clear, decision-ready format.

## Triggers

- Manual invocation

## Inputs

| Name   | Type   | Required | Description                                         |
| ------ | ------ | -------- | --------------------------------------------------- |
| data   | string | yes      | Raw data, metrics, or description of the dataset    |
| goal   | string | no       | Specific question or objective for the analysis     |
| output | string | no       | Desired output (summary, report, chart description) |

## Outputs

An analysis containing:
- Key findings and patterns
- Statistical highlights or comparisons
- Actionable recommendations
- Caveats or data-quality notes (if applicable)
