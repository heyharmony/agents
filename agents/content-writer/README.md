# Content Writer Agent

Writes polished content from a topic or brief — blog posts, social copy, newsletters, and more.

## Usage

Describe what you need written and optionally set `format`, `tone`, and `audience` to shape the output.

## Examples

- "Write a LinkedIn post about our new product launch" + tone: professional
- "Draft a blog intro on the future of remote work" + format: blog + audience: tech leaders
- "Create a short newsletter blurb announcing a webinar" + tone: friendly
